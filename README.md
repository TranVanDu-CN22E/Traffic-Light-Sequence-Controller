# Smart Traffic Light Control System

A multi-platform IoT system for controlling and monitoring a four-way traffic light intersection. The system supports multiple control methods, including physical buttons, Bluetooth, TCP/IP over LAN, and cloud synchronization through Firebase, allowing seamless interaction across Android, iOS, Windows Forms, and Web applications.

---

# Overview

This project simulates a smart traffic light controller for a four-way intersection using **ESP32** and a **Python Bridge Server** as the communication gateway.

Unlike traditional standalone traffic light systems, every platform communicates through a centralized bridge, enabling real-time synchronization between hardware devices and software applications.

---

# Features

## Traffic Light Control

* Turn system ON/OFF
* Normal operating mode
* Flash mode
* Manual control mode
* Individual route control
* Real-time traffic light status monitoring

---

## Supported Control Methods

* Physical push buttons
* Windows Forms application (TCP/IP LAN, Bluetooth)
* Android application (Firebase, Bluetooth)
* iOS application (Firebase, Bluetooth)
* Web application (Firebase)

# Communication Flow

### 1. Cloud Control

```text
Android / iOS / Web / Windows Forms
          │
          ▼
 Firebase Realtime Database
          │
          ▼
 Python Bridge Server
          │
          ▼
     ESP32 / Proteus
```

---

### 2. Windows Forms Control

```text
Windows Forms
      │
 TCP Socket (LAN)
      │
      ▼
Python Bridge Server
      │
      ▼
 ESP32 / Proteus
```

---

### 3. Bluetooth Control

```text
Android / iOS / Windows Forms
        │
Bluetooth Serial
        │
        ▼
Python Bridge Server
        │
        ▼
 ESP32 / Proteus
```

### 4. Physical Button Control

```text
Physical Buttons
        │
        ▼
```

# Activity Logging

Every event generated by the traffic light controller is automatically recorded.

Examples:

* System Started
* System Stopped
* Flash Mode Enabled
* Normal Mode Enabled
* Route Changed
* Manual Commands

Logs are stored in **Cloud Firestore** with timestamps for later viewing.
